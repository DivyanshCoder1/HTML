Guide for using TIMER:-
1. 'S' is the shortcut key to start timer.(Once timer starts then the start button is disabled).
2. 'SPACEBAR KEY' is the shortcut key to pause timer.(Works only after timer starts).
3. 'R' is the shortcut key to resume timer.(Works only when timer is pause).
4. 'ENTER KEY' is the shortcut key to restart timer.(It only works when the timer is started).
